# بوت تحميل يوتيوب - Telegram Bot

بوت تيليجرام لتحميل فيديوهات يوتيوب بجودات مختلفة باستخدام yt-dlp.

## الميزات

- 🔍 **بحث عن فيديوهات** - ابحث مباشرة من البوت بدون نسخ الرابط
- 📥 تحميل فيديوهات يوتيوب بجودات متعددة (144p إلى 1080p)
- 🎵 تحميل الصوت فقط (MP3)
- 📊 عرض معلومات الفيديو (العنوان، المدة، المشاهدات)
- 🖼️ عرض صورة مصغرة للفيديو
- ⌨️ أزرار تفاعلية لاختيار الجودة
- 🗑️ حذف تلقائي للملفات بعد الإرسال
- 🗄️ تخزين Supabase للإحصائيات والمستخدمين
- 🔧 لوحة تحكم للمطور
- 🚫 حظر المستخدمين
- 📢 بث رسائل لجميع المستخدمين
- 🇸🇦 واجهة عربية بالكامل

## المتطلبات

- Node.js (v14 أو أحدث)
- npm أو yarn
- yt-dlp (لتحميل الفيديوهات)
- FFmpeg (لمعالجة الصوت والفيديو)
- مفتاح بوت تيليجرام

## التثبيت

### 1. تثبيت Node.js
- حمّل من: https://nodejs.org/
- تحقق: `node -v`

### 2. تثبيت yt-dlp
```bash
# عبر pip
pip install yt-dlp

# أو حمّل يدوياً من:
# https://github.com/yt-dlp/yt-dlp/releases
```

### 3. تثبيت FFmpeg
- حمّل من: https://www.gyan.dev/ffmpeg/builds/
- استخرج الملف وأضف مجلد `bin` للـ PATH

### 4. الحصول على مفتاح البوت
- افتح تيليجرام وابحث عن `@BotFather`
- أرسل الأمر `/newbot`
- اتبع التعليمات وانسخ المفتاح

### 5. تثبيت المكتبات
```bash
npm install
```

### 6. إعداد البيئة
- أعد تسمية `.env.example` إلى `.env`
- استبدل `your_bot_token_here` بمفتاح البوت الفعلي
- (اختياري) أضف مفاتيح Supabase

### 7. التشغيل عبر Docker
```bash
docker build -t downloader-bot .
docker run --rm --env-file .env downloader-bot
```

ملاحظات Docker:
- يلتقط الحاوية `BOT_TOKEN` و`SUPABASE_URL` و`SUPABASE_ANON_KEY` و`SUPABASE_STORAGE_BUCKET` من ملف `.env`
- إذا كنت تحتاج مسار FFmpeg مخصصًا، ضع `FFMPEG_PATH` في ملف البيئة
- `npm start` هو الأمر الافتراضي داخل الحاوية، ويمكن استبداله عند التشغيل إذا لزم الأمر

## الأوامر

| الأمر | الوصف |
|-------|-------|
| `/start` | تشغيل البوت وعرض رسالة الترحيب |
| `/help` | عرض دليل المساعدة |
| `/search <كلمة>` | البحث عن فيديو في يوتيوب |
| `/audio` | تفعيل وضع الصوت فقط |
| `/video` | العودة لوضع الفيديو |
| `/admin` | لوحة تحكم المطور |
| `/stats` | عرض الإحصائيات (للمطور) |
| `/ban <user_id>` | حظر مستخدم (للمطور) |
| `/unban <user_id>` | إلغاء حظر (للمطور) |
| `/broadcast <رسالة>` | بث رسالة للجميع (للمطور) |

## طريقة الاستخدام

### التحميل برابط:
1. ابدأ البوت بالأمر `/start`
2. أرسل رابط فيديو يوتيوب
3. اختر الجودة المناسبة
4. انتظر حتى يتم التحميل

### البحث والتحميل:
1. استخدم الأمر `/search كلمة البحث`
2. اختر الفيديو من النتائج
3. اختر الجودة المناسبة
4. انتظر حتى يتم التحميل

## القيود

- الحد الأقصى لمدة الفيديو: 20 دقيقة
- الحد الأقصى لحجم الملف: 50 ميجابايت (حد API تيليجرام)
- بعض الفيديوهات قد تكون محظورة حسب المنطقة

## حل المشاكل

**البوت لا يستجيب:**
- تأكد من صحة مفتاح البوت في `.env`
- تحقق من تثبيت Node.js والمكتبات
- راجع رسائل الخطأ في الـ console

**التحميل يفشل:**
- الفيديو قد يكون خاص أو محظور بالعمر أو المنطقة
- حجم الملف قد يتجاوز 50 ميجا
- جرّب جودة أقل

**yt-dlp لا يعمل:**
- تأكد من تثبيته: `yt-dlp --version`
- تأكد من أنه متاح في PATH

**FFmpeg لا يعمل:**
- تأكد من تثبيته: `ffmpeg -version`
- تأكد من إضافة مجلد `bin` للـ PATH
